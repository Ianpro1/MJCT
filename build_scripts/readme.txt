If the user intent to use this environment repository to develop tasks and create his own python bindings,
he can simply run the first_build.bat file.
It will install all the dependencies and create a build folder.
The mjct module will be located in the build/debug directory.

To update the project further, the user can run the build.bat file.

Furthermore, the debug.bat, test.bat and test_mjct.bat allow the user to quickly open a cmd prompt in the debug folder (goto_debug),
as well as run python (test.bat) and import the mjct module (test_mjct.abt).
